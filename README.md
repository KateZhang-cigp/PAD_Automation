\# PAD Automation Project



\## Overview

This project automates PAD (Personal Account Dealing) control checks.



\## Features

\- Match broker trades with PAD approvals

\- Generate Excel report

\- Monthly tracking



\## Files

\- Futu\_Files, HSBC\_Files, IBKR\_Files -> data sources

\- Combined\_Broker\_Trades.xlsx -> input trades

\- PAD\_Control\_Result.xlsx-> output result



\## How to run

Run the jupyter script:PAD\_Statement Extraction.ipynb
